# Talha BOYALIKLI Personal Website

A modern, responsive personal website for Talha BOYALIKLI, a maritime operations and finance professional.

## Features

- **Modern Dark Theme**: Professional dark interface with blue accents
- **Fully Responsive**: Works on mobile, tablet, and desktop
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Mobile hamburger menu
  - Animated timeline and cards
  - Contact form (frontend only)
- **Sections**:
  - Hero with contact information
  - Career objective
  - Professional experience timeline
  - Skills & expertise (maritime operations, software, languages, certifications)
  - Education
  - Contact form and details

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Poppins, Roboto)

## Deployment

This website is deployed using GitHub Pages at: [https://talhaboy1.github.io](https://talhaboy1.github.io)

## Local Development

To run this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/talhaboy1/talhaboy1.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd talhaboy1.github.io
   ```

3. Open `index.html` in your web browser.

## Customization

To customize this website for your own use:

1. Update personal information in `index.html`
2. Modify color scheme in `style.css` (CSS variables at the top)
3. Replace placeholder content with your own
4. Update the favicon in the `<head>` section
